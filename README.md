# VeckPRIV — Privacy-First Web Tools & Arcade

> **Your data never leaves your browser.**

VeckPRIV is an independent project that brings together a collection of web tools and lightweight arcade games that run 100% client-side. No data is ever sent to a server — no tracking, no ads, no accounts required.

## ❓ What is VeckPRIV?

VeckPRIV was born out of frustration with the modern web ecosystem — one filled with tracking, data harvesting, and heavy reliance on third-party services. We believe users deserve digital tools that are fast, transparent, and genuinely private.

Every tool we build:
- Runs entirely in your browser
- Sends zero data to any server
- Uses no cookies, analytics, or telemetry
- Requires no sign-up or login
- Works instantly and is free forever

## 🧭 Philosophy

VeckPRIV is more than just a collection of tools — it's a statement that the web can function beautifully without compromising user privacy. We stand for:

- **Absolute privacy**: no tracking, no user profiling
- **Transparency**: code that anyone can inspect
- **Independence**: no reliance on cloud services or third-party vendors
- **Simplicity**: lightweight, fast, and easy to use
